# Documentation

Start here to understand what the IEC Common Data Dictionary is, how the
OpenCDD Browser is organised, and how to consume CDD data in code. The
browser lives at
[**opencdd.github.io**](https://opencdd.github.io/) and is free to use.

Throughout these docs, quoted definitions and section references come
from the IEC 61360 series. The IEC publications are © IEC, Geneva; we
paraphrase them here and point you to the
[IEC webstore](https://webstore.iec.ch/) for the canonical text.
